À propos

Webcitation est une application regroupant des citations de personnes travaillant dans l’industrie du web. Elle est conçue, construite et mise à jour par David Leuliette, pour vous inspirer toutes les semaines. Il réalise une sélection des meilleures citations croisées sur différents articles de blogs, livres ou conférences.

Contributions

Seulement les citations en Français sont acceptées. Les citations doivent avoir un minimum de rapport avec la création et le développement de sites / applications web (Même si leur auteur n’a pas connu les internets, comme Léonard de Vinci). Les propositions avec un contenu inapproprié ne sont pas acceptées.

Comment contribuer ?

Pour partager une citation, lisez le README ou créez une pull request.
